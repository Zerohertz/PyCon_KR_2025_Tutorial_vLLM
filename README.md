<h3 align="center">How to Setup?</h3>

1. [`uv`](https://docs.astral.sh/uv/) 설치

```bash
curl -LsSf https://astral.sh/uv/install.sh | sh
```

2. 의존성 설치

```bash
uv sync
```

<h3 align="center">How to Run?</h3>

```bash
uv run ${PYTHON_SCRIPT}.py
```
